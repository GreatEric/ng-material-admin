# ng-material-admin

基于 Angular8 做的一个后台内容管理模板，运行该项目的时候确保当前环境安装了[Angular CLI](https://github.com/angular/angular-cli) 8.3 以上版本的，

## Development server && Build

```
// 如果没有安装 Angular CLI
npm install -g @angular/cli

// 开发
ng serve --open

// 构建
ng build
```
